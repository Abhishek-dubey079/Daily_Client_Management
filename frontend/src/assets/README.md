# Assets Folder

## Logo Image

Please place your logo image file here:

**File name:** `app_logo.png`
**Location:** `frontend/src/assets/app_logo.png`

**Requirements:**
- Format: PNG (recommended) or JPG
- Recommended size: 512x512px (square)
- Will be displayed as circular with object-fit: cover

The logo will be used in:
- Navbar/Header
- Login page
- Dashboard
- Favicon (via public folder)
